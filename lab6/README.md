# Lab 6 - DOM Manipulation + Express Routes

This project contains all 7 tasks from the lab sheet.

## Run

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start server:
   ```bash
   npm start
   ```
3. Open:
   `http://localhost:3000`

## Task Mapping

- Task 1: `/task1-gallery.html`
- Task 2: `/task2-quiz.html`
- Task 3: `/task3-kanban.html`
- Task 4: `/task4-table-generator.html`
- Task 5: `/task5-nested-table.html`

  - `GET /products`

- Task 7:
